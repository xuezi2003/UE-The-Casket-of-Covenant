# 关卡 1 场景组件

本文件夹包含关卡 1 中使用的场景组件蓝图文档。

## 组件列表

| 组件 | 用途 | 状态 |
|------|------|:----:|
| [BP_ArenaGenerator](组件/BP_ArenaGenerator.md) | 场地生成器，动态生成完整关卡结构 | ❌ |
| [BP_BornVol](组件/BP_BornVol.md) | 出生体积，提供随机玩家出生点 | ✅ |
| [BP_Camera](组件/BP_Camera.md) | 壁挂监控，增强"监视"氛围 | ✅ |
| [BP_BlackBoardWithPaint](组件/BP_BlackBoardWithPaint.md) | 展示随机粉笔画的单体黑板 | ✅ |
| [BP_BlackboardWall](组件/BP_BlackboardWall.md) | 黑板墙管理器，驱动交换动画 | ✅ |
| [BP_Obstacle](组件/BP_Obstacle.md) | 随机生成课桌/椅子的障碍物 | ✅ |

### ArenaGenerator 子模块（❌ 待实现）

| 子模块 | 用途 |
|--------|------|
| BP_Section_Start | 起点封闭墙（包含 BP_BornVol） |
| BP_Section_End | 终点封闭墙（包含 BP_VideoBoard + BP_FinishLine） |
| BP_Section_Wall | 单侧墙段（包含 BP_BlackboardWall） |
| BP_Section_Pillar | 单侧柱段（包含 BP_Camera） |
| [BP_VideoBoard](组件/BP_VideoBoard.md) | 终点视频黑板（不同情况播放不同视频） |
| [BP_FinishLine](组件/BP_FinishLine.md) | 终点线触发器 |

> [!NOTE]
> 子模块详细说明见 [BP_ArenaGenerator.md](组件/BP_ArenaGenerator.md)。
>
> 障碍物散布使用 **IA Scatter** 插件实现，不使用 PCG。
