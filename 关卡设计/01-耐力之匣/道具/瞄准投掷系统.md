# 瞄准投掷系统

**父类**：GSCGameplayAbility

## 设计概述

瞄准投掷系统允许玩家使用投掷类道具攻击其他玩家。

**核心机制**：
- 按住右键进入瞄准状态，松开取消
- 瞄准中按左键投掷道具
- 投掷后自动结束瞄准状态
- 手持道具通过 Multicast RPC 同步显示

**数据资产**：详见 [DA_ItemSystemConfig.md](DA_ItemSystemConfig.md)

## GA_Aim（瞄准能力）

**路径**：`Content/0_/Level/1_Endurance/GAS/GA/GA_Aim`

### 标签配置

| 配置项 | 值 |
|--------|-----|
| 资产标签 | `Player.Action.Aiming` |
| 激活阻止标签 | `Aiming`, `Throwing`, `Fallen`, `Staggered` |

### 高级配置

| 配置项 | 值 |
|--------|-----|
| 网络执行策略 | 本地预测 |
| 实例化策略 | 每个 Actor 实例化 |
| 重新触发实例化能力 | ✅ |

### 变量

| 变量名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| ConfigAsset | DA_ItemSystemConfig | - | 数据资产引用 |
| Effect Container | Array\<Handle\> | - | GE Handle 缓存 |
| ItemData | S_ItemData | - | 缓存的道具数据（Check 宏填充） |
| BP_TrajectoryDisplay | BP_TrajectoryDisplay | - | 预测线 Actor 引用 |
| PredictTimerHandle | Timer Handle | - | 预测线更新定时器句柄 |
| UpRate | Float | 0.2 | 投掷上扬系数 |
| RightRate | Float | 0.1 | 投掷右偏系数（避开角色遮挡） |

### 事件图表

```
Event ActivateAbility
    ↓
Check Have Throwable (宏)
    └─ GetThrowItemID → Is Valid? → 查表填充 ItemData
    ├─ False → End Ability
    └─ True ↓
CommitAbility → Apply Effect Container (AimEffectContainer) → SET Effect Container
    ↓
UseYaw(true)  ← 开启角色跟随镜头旋转
    ↓
ShowSMProp(true)  ← 显示手持道具
    ↓
ShowTrajectory()  ← Spawn 预测线 Actor
    ↓
SetTimerDelegate (UpdateTrajectory, 0.033, Looping) → SET PredictTimerHandle
    ↓
Wait Input Release (IA_Endurance_Aim)
    │
    ├─【On Release】→ End Ability
    │
    └─【同时】Wait Gameplay Event (Gameplay.Event.Aim.End)
            ↓ Event Received
        End Ability  ← 投掷完成时由 GA_Throw 发送事件触发

---
Event OnEndAbility
    ↓
UseYaw(false)  ←以此恢复默认旋转模式
    ↓
ShowSMProp(false)  ← 隐藏手持道具
    ↓
ClearAndInvalidateTimerHandle(PredictTimerHandle)
    ↓
HideTrajectory()  ← 销毁预测线 Actor
    ↓
For Each (Effect Container) → RemoveGEFromOwnerWithHandle
    ↓
Array Clear
```

> [!NOTE]
> **双重结束机制**：GA_Aim 可通过两种方式结束：
> 1. 玩家松开右键（Wait Input Release）
> 2. 投掷完成后收到 `Gameplay.Event.Aim.End` 事件（由 GA_Throw 发送）

### ShowSMProp(IsShow) 函数

```
Has Authority?
    └─ True ↓
Get Avatar Actor → Cast to BP_Character_Game
    ↓
GetComponentByClass (Comp_Character_Endurance)
    ↓
Call Multicast_ShowPropInHand(IsShow)
```

### UseYaw(UseYaw: Bool) 函数

```
Cast to BP_Character_Game
    ↓
Set UseControllerRotationYaw = UseYaw
    ↓
CharacterMovement.SetOrientRotationToMovement = !UseYaw
```

### 预测线相关函数

#### ShowTrajectory ✅

```
ShowTrajectory
    ↓
NOT HasAuthority?  ← 只在客户端执行
    └─ True ↓
Cast to BP_Character_Game
    ↓
SpawnActor (BP_TrajectoryDisplay, Instigator=Character)
    ↓
SET BP_TrajectoryDisplay
    ↓
BP_TrajectoryDisplay.ShowTrajectory(true)
```

#### UpdateTrajectory ✅

```
UpdateTrajectory
    ↓
IsValid(BP_TrajectoryDisplay)?
    └─ True ↓
Cast to BP_Character_Game
    ↓
BP_TrajectoryDisplay.UpdateTrajectory(
    LaunchVelocity = GetVel(),
    StartPos = Mesh.GetBoneTransform(Socket_Throw).Location
)
```

#### HideTrajectory ✅

```
HideTrajectory
    ↓
IsValid(BP_TrajectoryDisplay)?
    └─ True ↓
BP_TrajectoryDisplay.ShowTrajectory(false)
    ↓
BP_TrajectoryDisplay.DestroyActor()
    ↓
SET BP_TrajectoryDisplay = None
```

#### GetVel ✅

```
GetVel
    ↓
Cast to BP_Character_Game → FollowCamera
    ↓
Return: ItemData.LaunchForce × Normalize(
    FollowCamera.ForwardVector +
    WorldUp(0,0,1) × UpRate +
    FollowCamera.RightVector × RightRate
)
```

> [!NOTE]
> 投掷方向跟随摄像机视角，`UpRate` 提供上扬，`RightRate` 向右偏移避开角色遮挡（右手投掷）。

## GA_Throw（投掷能力）

**路径**：`Content/0_/Level/1_Endurance/GAS/GA/GA_Throw`

### 标签配置

| 配置项 | 值 |
|--------|-----|
| 资产标签 | `Player.Action.Throwing` |
| **激活所需标签** | `Player.Action.Aiming` |
| 激活阻止标签 | `Throwing`, `Fallen`, `Staggered` |

### 高级配置

| 配置项 | 值 |
|--------|-----|
| 网络执行策略 | 服务器已启动 |
| 实例化策略 | 每个 Actor 实例化 |
| 重新触发实例化能力 | ✅ |

### 变量

| 变量名 | 类型 | 说明 |
|--------|------|------|
| ConfigAsset | DA_ItemSystemConfig | 数据资产引用 |
| Effect Container | Array\<Handle\> | GE Handle 缓存 |
| ThrowItemID | DataTableRowHandle | 缓存的道具 ID |
| ItemData | S_ItemData | 缓存的道具数据（ActivateAbility 时查表填充） |
| ThrowItem | BP_Item_Base | 生成的道具引用 |
| UpRate | Float | 投掷上扬系数 |
| RightRate | Float | 投掷右偏系数 |

### 事件图表

```
Event ActivateAbility
    ↓
Cast to BP_Character_Game → GetThrowItemID → SET ThrowItemID
    ↓
Get Data Table Row (DT_ItemData, ThrowItemID) → SET ItemData  ← 查表缓存
    ↓
CommitAbility → Apply Effect Container (ThrowEffectContainer) → SET Effect Container
    ↓
PlayMontageAndWaitForEvent (Anim_Throw_Montage)
    │
    ├─【Event Received】(Gameplay.Event.Montage.Throw)
    │       ↓
    │   Switch Has Authority
    │       ↓ Authority
    │   SpawnAndThrow → End Ability
    │
    ├─【On Cancelled】 → End Ability
    │
    └─【On Interrupted】 → End Ability

---
Event OnEndAbility
    ↓
SendGameplayEventToActor (Gameplay.Event.Aim.End)  ← 通知 GA_Aim 结束
    ↓
For Each (Effect Container) → RemoveGEFromOwnerWithHandle
    ↓
Array Clear
```

### SpawnAndThrow 函数

```
SpawnAndThrow
    ↓
SpawnItem() → SET ThrowItem
    ↓
ThrowItem.ProjectileMovement.Velocity = GetVel()
    ↓
ClearThrowItemID
```

**辅助函数**：

| 函数 | 返回类型 | 逻辑 |
|------|----------|------|
| `SpawnItem` | BP_Item_Base | 见下方详细实现 |
| `GetVel` | Vector | 与 GA_Aim.GetVel 一致（使用缓存的 ItemData.LaunchForce） |

### SpawnItem 函数 ✅

```
SpawnItem
    ↓
Cast GetAvatarActorFromActorInfo() to BP_Character_Game
    ↓
Spawn Actor (ItemData.ItemClass)  ← 动态类
    ├─ Transform: Mesh.GetBoneTransform(Socket_Throw, RTS_World)
    ├─ CollisionHandling: Default
    ├─ Instigator: Character (投掷者)
    ├─ ItemID: ThrowItemID
    └─ InitialState: Flying
    ↓
Return SpawnedActor
```

> [!NOTE]
> **动态类生成**：使用 `ItemData.ItemClass` 而非硬编码 BP_Item_Base，确保生成对应的子类（如 BP_Item_Banana）。

## 动画蓝图集成 (ABP_Character)

### 变量

| 变量名 | 类型 | 用途 |
|--------|------|------|
| bIsAiming | Boolean | 瞄准状态标记 |

### Event Graph

```
Event Blueprint Update Animation
    ↓
Get Ability System Component
    ↓
Has Matching Gameplay Tag (Player.Action.Aiming)
    ↓
SET bIsAiming
```

### AnimGraph 配置

```
Main States (Cache)
    ↓
Slot 'DefaultSlot' ──────┐
                         ↓ (Base)
                    Layered Blend Per Bone  ←── [Weight: bIsAiming (Smoothed)]
                         ↑ (Blend Pose 0)
Anim_Aim (Seq) ────→ Slot 'UpperBody'
```

**Layered Blend Per Bone 设置**：

| 配置项 | 值 |
|--------|-----|
| Branch Filter | `spine_01` |
| Blend Depth | 1 |
| Mesh Space Rotation Blend | ✅ |

**平滑处理**：使用 `FInterpTo` 对 bIsAiming 进行插值（Interp Speed ≈ 10）
