# UE5 开发技巧：硬核排查资产引用丢失

当资产报错引用丢失，但在 Details 面板找不到对应项时，使用以下方案：

### 核心步骤

1. **获取底层属性名**：
   - 在内容浏览器选中资产，`Ctrl + C`。
   - 粘贴至文本编辑器，`Ctrl + F` 搜索报错的路径/文件名。
   - 记录该路径**等号左侧的变量名**（即底层属性名）。

2. **属性矩阵修改**：
   - 右键资产 -> **Asset Actions** -> **Property Matrix**。
   - 在矩阵设置中搜索刚才记录的变量名。
   - 直接在此处修正引用路径。

### 原理
UE 资产支持文本化。通过文本编辑器绕过 UI 限制获取“真实变量名”，再利用属性矩阵（Property Matrix）直接操作资产底层数据结构。
