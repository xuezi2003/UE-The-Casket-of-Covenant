# 关卡5 牺牲之匣 (L_Sacrifice)

## 核心玩法

- **场景**：悬空玻璃桥，多对玻璃板
- **视角**：第三人称
- **核心**：观察、测试、博弈

## 玻璃板机制

| 类型 | 承重 | 说明 |
|:-----|:-----|:-----|
| 安全玻璃 | ≤2人 | >2人破碎 |
| 普通玻璃 | 0人 | 踏上即碎 |

## 观察系统

- **桥两侧**：可通过外观区分安全/普通玻璃
- **桥中心**：外观区分失效（材质反射相同）

## 金币测试

- 消耗金币扔硬币到前方玻璃
- 不同玻璃发出不同声音

## 移动规则

- 自由移动，无固定顺序
- 玻璃碎了就是碎了（留空）

## 玩家交互

- 推搡/失衡/摔倒/闪避机制参照关卡1
- **限制**：只有同一块玻璃上的玩家才能互相推搡
- QTE 失败 → 直接摔下去（死亡）

## 开始/结束条件

**开始条件**：所有玩家加载完成，AI 最先生成

**结束条件**（满足任一）：
- 存活玩家到达终点
- 全员死亡
- 真人全死（只剩AI）→ 结束整局
- 整体时间限制

## 策略要素

- 两人同站安全玻璃 → 安全
- 第三人加入 → 全部掉落
- 推搡可把人推到未知玻璃上测试
- 金币测试前方路径
