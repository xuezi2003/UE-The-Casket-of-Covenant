# 关卡3 勇气之匣 (L_Courage)

## 核心玩法

- **场景**：站台 + 四节车厢列车
- **视角**：横版第三人称（车厢侧面）
- **核心**：线索收集 → 推理 → 生死抉择

## 游戏流程

### 阶段一：站台（线索收集）

- **时长**：约1.5分钟
- 散落线索：海报、杂志、报纸等
- 物品检视：3D旋转查看
- 金币购买：部分线索需花费
- 收集的线索存入"背包"，车上可查看（仅自己可见）

### 阶段二：上车

- **时长**：30秒内必须上车，否则死亡
- 四节车厢，每节有对应线索
- 玩家可投票"有毒/没毒"（信息交流用）

### 阶段三：毒气判定（逐节进行）

```
每节车厢：
1. 视线遮蔽（窗户不透明，看不到他人）
2. 倒计时开始
3. 玩家自由戴/摘面具
4. 毒气生效判定：
   - 安全车厢 → 无事
   - 毒气车厢 + 戴面具 → 消耗1个解毒剂，存活
   - 毒气车厢 + 没戴面具 → 死亡
5. 进入下一节车厢
```

## 开始/结束条件

**开始条件**：所有玩家加载完成，AI 最先生成

**结束条件**（满足任一）：
- 存活玩家到达列车尾部
- 全员死亡
- 真人全死（只剩AI）→ 结束整局
- 整体时间限制

## 核心资源

| 资源 | 数量 | 说明 |
|:-----|:-----|:-----|
| 面具（解毒剂） | 3个 | 毒气判定时消耗 |
| 毒气车厢 | 3节 | 需要面具或推理规避 |
| 安全车厢 | 1节 | 无需面具 |

## 移动规则

- 站台/车厢内可自由移动（横版）
- 只能在当前车厢内活动

## 惩罚机制

- 未在30秒内上车 → 直接死亡

## AI 行为

- AI 知道当前车厢是否有毒
- 按概率决策戴/摘面具
