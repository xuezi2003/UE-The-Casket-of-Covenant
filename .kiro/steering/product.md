# 产品概述

《契约之匣》是一款多人在线生存竞技游戏，运行在 Unreal Engine 5 的 Dedicated Server 上。

## 核心概念

- 玩家需通过 5 个关卡考验：耐力之匣、逻辑之匣、勇气之匣、洞察之匣、牺牲之匣
- 支持真人玩家与 AI 混合参赛（目标人数约 10-15 人）
- 淘汰制：HP 归零即淘汰，无复活机制
- 掉线玩家由 AI 接管继续游戏

## 关卡简介

| 关卡 | 核心玩法 |
|------|----------|
| 耐力之匣 | 红绿灯机制，木偶监视，静止/移动判定 |
| 逻辑之匣 | 答题博弈，安全/危险房间选择 |
| 勇气之匣 | 线索收集，毒气车厢推理 |
| 洞察之匣 | 无限走廊，异常观察，站队判断 |
| 牺牲之匣 | 玻璃桥，承重判断，推搡博弈 |

## 设计风格

灵感来源于《弥留之国的爱丽丝》《鱿鱼游戏》等生存竞技作品。
